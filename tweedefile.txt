git clone "linkrepo"
-------------------------------------------------------------
naar branch switchen: git checkout -b "naambranch"
Om aanpassingen naar bracnh te pushen:
-git add
-git commit
-git push
Om van branch te pullen:
-git fetch
-git pull origin "naambranch"
-----------------------------------------------------------
om specifieke branch te clonen:
git clone -b "naamBranch" "linkrepo"
-----------------------------------------------------------
om te pushen naar branch:
git push origin "naamBranch"
----------------------------------------------------------
om commit te verwijderen dat nog niet gepushed werd:
git reset HEAD~1
